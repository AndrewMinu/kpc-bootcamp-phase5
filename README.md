# KPC 부트캠프(로컬 싱크) — 5단계: 피칭덱

## 설치 (받자마자 한 번)
```bash
git clone <이 레포 주소> && cd kpc-bootcamp-phase5   # 또는 zip 다운로드
bash install.sh     # ~/.agents/skills 에 이 단계 스킬 추가(어디서든)
```
Codex에서 `/skills` 확인 → `$bootcamp-guide` 로 이어서 진행.
진행 상태(context.md)는 Codex 작업 폴더에 남아 단계가 바뀌어도 그대로 이어집니다.

1단계(kpc-bootcamp-phase1)를 먼저 설치했어야 합니다.
