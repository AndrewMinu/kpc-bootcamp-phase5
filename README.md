## 설치 — 한 줄 (권장, npx)
```bash
npx skills@latest add <내-github-아이디>/kpc-bootcamp-phase5 -a codex -g -y
```
- 그다음 Codex에서 `/skills` 확인 → `$bootcamp-start` 로 시작.
- 진행 상태(context.md)는 Codex 작업 폴더에 남아 단계가 바뀌어도 이어집니다.
